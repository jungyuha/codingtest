# 1. 시간 복잡도

## \[1] log(n) 의 의미

* 다음 코드의 시간 복잡도는 **O(nlog(n))**이다.

```cpp
int i, j, k = 0;
for (i = n / 2; i <= n; i++) { // O(n)
    for (j = 2; j <= n; j = j * 2) { // O(log(n))
        k = k + n / 2;
    }
}
```

* 다음 코드의 시간 복잡도는 **O(log(n))**이다.

```cpp
int a = 0, i = N;
while (i > 0) {
    a += i;
    i /= 2;
}
```
