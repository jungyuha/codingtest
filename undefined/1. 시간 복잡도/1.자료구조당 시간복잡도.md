# 1.자료구조당 시간복잡도

## \[1] 자료구조 시간복잡도

<figure><img src="../../.gitbook/assets/image (3) (4).png" alt=""><figcaption></figcaption></figure>

### \[1] Array

* 접근 : O(1)
* 검색 : O(n)
* 삭제/수정 : O(n)

### \[2] Stack

* 접근 : O(n)
* 검색 : O(n)
* 삭제/수정 : O(1)

### \[3] 레드 블랙 트리 (맵) : O(nlog(n))

### \[4] STL&#x20;

* Sort : O(nlog(n))
* find : O(n)

## 공간복잡도

* 보통 n을 1000만 이하로 한다.
* 보통 mn을 10억 이하로 한다.
