## \[1] 이론

### (1) 비트마스킹

### (2) 내가 헤멘 이유

* 성곽의 벽을 허물 때 1칸의 벽만 허물었다.
  * 성곽의 벽은 총 2개의 칸에서 허물어야했다.
* 완전 탐색이 아니라 그냥 dfs 문제였다.
* vis 초기화를 제대로 안 해줬다.

#### 내 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int a,res;
int main(){
    cin >> a;
    for(int i=0;i<7;i++){
        if(a & (1<<i)){
            res++;
        }
    }
    cout << res << "\n";
    return 0;
}
```
